# Numora · Calculadoras laborales

Web estática en HTML, CSS y JavaScript puro.

## Incluye

- Home de Numora.
- 8 calculadoras laborales para España.
- Diseño responsive y rápido.
- SEO básico: canonical, sitemap, robots y schema.
- Consent Mode por defecto en denegado.
- Banner de consentimiento corregido.
- Espacios preparados para AdSense.
- Páginas legales básicas.

## Antes de publicar

1. Comprueba que el dominio definitivo sea `https://www.numora.es` o cambia todas las URLs si usas otro.
2. Completa datos reales en `aviso-legal.html` y `contacto.html`.
3. Para AdSense en la UE, configura una CMP certificada por Google antes de servir anuncios personalizados.
4. Sustituye los bloques “Espacio para AdSense” por el código real cuando AdSense esté aprobado.

## Publicación rápida

Sube todos los archivos a GitHub y despliega en Vercel como proyecto estático.
